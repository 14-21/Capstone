Introduction

Welcome to the Game Review API hosted by Colton Skiera, Kelsey Gazaway, Sarah Schultz, and Jessie Gipson. This API is a first template API created for use with pulling game information such as, Game Title, Studio, Description, Platform, and more. Please refer to the paths section to find the correct paths to the information you are looking for.

Fetch All Games Data

  • GET /games

A request made to this endpoint will "fetch" an array of objects containing all data held for each game.

Fetch Options

Headers

Content-Type(String, if using headers required):application/json

Return Data

  • game(array of objects)
  • id(string): This is the database identifier for the post object.
  • title:(string) This is the name of the Game.
  • platform:(string) This is the list of different game systems that the game can be played on.
  • genre:(string) This is the type/style of the game.
  • studio:(string) This is the name of the manufacturers/creators of the game.

Fetch All Users Data

  • GET /games/users

    • A request to this endpoint will "fetch" all users data.

  • GET /games/usernames

    • A request to this endpoint will "fetch" a specific user by their username. This requires an admin login.

  • GET /adminusers

    • This endpoint will list all Admin Users. Requires Admin to use.

  • GET /games/get/user.

    • This endpoint will "fetch" a user by their SERIAL PRIMARY KEY in the database.

Fetch All Reviews Data

  • GET /api/games/reviews

    • This endpoint will "fetch" all the reviews on the reviews table of the database.

  • GET /api/games/user/specific/reviews

    • This endpoint will "fetch" all reviews made by a designated user.

Get Comments Data

  • GET /games/users/comments/:origReviewId

    • This endpoint will "fetch" all comments by the review it is posted to.

  • GET /api/user/review/comments

    • This endpoint "fetches" all comments made by a specified user.

  • GET /api/games/all/comments

    • This endpoint "fetches" all comments from the database. This endpoint is only usable by and Admin User.

Posting to the Database

Listed below are all post routes usable on the API.

  • POST /games/create/game

    • You must be an admin to use this route.

  • POST /games/users/register

    • This Route is used to Sign up for an account on the website.

  • POST /games/users/login

    • This route allows registered users to login.

  • POST /api/review/post/comment

    • This endpoint allows a new comment to be posted by a logged in user.

  • POST /games/post/review

    • This endpoint allows logged in users to post a new review to a game.

Update Routes for the API

  • PUT /api/games/updategame

    • This route will allow Admins to update information on a specific game.

  • PUT /api/comments/update/:commentId

    • This route allows a user to edit a previously posted comment.

  • PUT /games/user/review/update

    • This route allows uses to edit a previously posted Game Review.

Delete Routes on the API

  • DELETE /api/games/delete/:gameId

    • This route will allow Admin users to delete a game.

  • DELETE /api/users/delete/:userId

    • This route allows Admin users to delete a user account.

  • DELETE /api/games/user/review/delete/:id

    • This route allows users to delete a review they have previously posted.

  • DELETE /api/games/comments/delete/:commentId

    • This route allows users to delete a previously posted comment.
